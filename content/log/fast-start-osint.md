Когда нужно быстро «пробить» никнейм, не нужно открывать браузер и десятки вкладок. Достаточно консоли Termux и трех проверенных инструментов.



### 1. Sherlock — Массовый охват
Первый шаг — проверка регистрации на сотнях популярных ресурсов. 
```bash
python sherlock nickname
```

### 2. Maigret — Глубокий анализ
Если Sherlock дает список ссылок, то Maigret собирает данные *внутри* аккаунтов: ищет ID, связанные почты и дополнительные имена.
```bash
maigret nickname
```

### 3. Nexfil — Скорость
Когда время ограничено, Nexfil проверяет основные соцсети за считанные секунды благодаря многопоточности.

### Мой вердикт
Для 90% задач этого набора более чем достаточно. Весь процесс занимает не более 5 минут и работает прямо в кармане.
