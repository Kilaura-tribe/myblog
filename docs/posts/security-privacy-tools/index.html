Цифровая гигиена: Инструменты анонимности

March 13, 2026

В OSINT безопасность — это не паранойя, а необходимость. Вот базовый набор инструментов для защиты своих данных:

1. Браузеры и расширения#

  • Mullvad Browser — проект от создателей Tor, настроенный на максимальную защиту от “отпечатков” браузера (fingerprinting).
  • Brave — хороший выбор для повседневного использования с вырезанной рекламой и трекерами.
  • uBlock Origin — обязательное расширение для любого браузера. Блокирует не только рекламу, но и скрытые скрипты слежки.

2. Почта и связь#

  • ProtonMail — зашифрованная почта из Швейцарии. Не требует номер телефона при регистрации через Tor.
  • SimpleLogin — сервис для создания почтовых алиасов. Позволяет не “светить” основную почту на сомнительных ресурсах.

3. VPN и прокси#

Для OSINT лучше использовать проверенные платные решения с открытым исходным кодом, такие как Mullvad VPN или IVPN. Они не ведут логи и позволяют выходить в сеть под другими IP.

4. Полезные инструменты в Termux#

Для проверки своей безопасности можно использовать:

  • dnslookup — чтобы проверить, не текут ли твои DNS-запросы.
  • proxychains — для пропуска любого трафика через цепочку прокси-серверов.

Помни: Никакой софт не спасет, если ты сам оставляешь свои данные в открытом доступе.
