Зачем использовать тяжелый ПК, если всё можно сделать на Android? В этом посте я расскажу, как этот блог был собран с нуля в консоли Termux.

### 🛠 Мой набор инструментов
Для работы мне понадобилось всего три вещи:
1. **Termux** — терминал для Android.
2. **Hugo** — генератор статических сайтов (быстрый и легкий).
3. **GitHub** — место, где сайт живет и откуда открывается миру.

### 📝 Основные этапы сборки
Весь процесс можно уместить в несколько команд.

1. **Установка софта:**
```bash
pkg update && pkg upgrade
pkg install hugo git```

2. **Создание проекта и выбор темы:**
Я выбрал тему PaperMod, так как она минималистичная, быстрая и идеально подходит для технических блогов.

3. **Настройка деплоя:**
Я настроил Hugo так, чтобы он собирал сайт в папку docs. Это позволило мне использовать бесплатный хостинг GitHub Pages без лишних сложностей.


   **​🚀 Почему это круто?**
​*Мобильность:* Я могу написать новый пост или поправить код, сидя в автобусе или кафе.
​*Бесплатно:* Я не плачу за хостинг ни копейки.
*​Скорость:* Сайт на Hugo загружается мгновенно даже при плохом интернете.

   **​Вывод:** Смартфон — это не только про соцсети, это полноценный инструмент для разработки.
