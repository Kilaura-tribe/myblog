Привет! В этом гайде мы пошагово разберем, как установить **Nexfil** — мощный инструмент для OSINT, который ищет аккаунты по никнейму более чем на 350 ресурсах.

Ниже приведен список команд. Вводи их в Termux по очереди.

---

### Инструкция по установке:

1. **Обновление системы** Эта команда обновляет список доступных программ и их версии:

   ```
pkg update && pkg upgrade -y```


2. Установка Git и Python Git нужен для скачивания кода с GitHub, а Python — для работы самой программы:

   ```
pkg install git python -y```


3. Загрузка программы Nexfil Копируем файлы инструмента из официального репозитория к себе на устройство:

   ```
git clone [https://github.com/thewhiteh4t/nexfil.git](https://github.com/thewhiteh4t/nexfil.git)```


4. Переход в рабочую папку Заходим в директорию, в которую только что скачали программу:

   ```
cd nexfil```


5. Установка необходимых библиотек Докачиваем специальные модули, которые нужны для корректной работы сканера:

   ```
pip install -r requirements.txt```


6. Как запустить поиск?
   
   ​Чтобы начать поиск по никнейму (например, ivan), используй следующую команду:
 
   ```
python3 nexfil.py -u ivan```


         После завершения поиска программа создаст файл с результатами прямо в папке nexfil. Удачных расследований!
