OSINT Ресурсы

🔍 Никнеймы и Почта#

  • Sherlock — Поиск по 400+ сайтам.
  • Maigret — Глубокий анализ по нику.
  • EPIEOS — Разведка по Email.

🇺🇦 Украина (Реестры)#

  • YouControl — Проверка ФОП/Компаний.
  • Opendatabot — Мониторинг данных.
  • Судова влада — Судебные дела.

💰 Криптовалюты (Blockchain OSINT)#

  • Blockchain.com — Эксплорер BTC/ETH.
  • WalletExplorer — Поиск связей между кошельками.
  • Chainabuse — База данных мошеннических адресов.

🛡 Утечки и Даркнет#

  • Have I Been Pwned — Проверка почты на утечки.
  • Snusbase — (Premium) Поиск по слитым базам.
  • Onion Engine — Поисковик по .onion сайтам.

📍 Геолокация#

  • SunCalc — Определение времени по теням.
  • PeakVisor — Идентификация гор по фото.

Note: Всегда используй разные браузерные профили при работе с этими ссылками.

🛡 Гигиена и Анонимность#

  • Mullvad VPN — Самый анонимный (не требует даже почту при регистрации).
  • Proton VPN — Надежный сервис из Швейцарии, есть хорошая бесплатная версия.
  • 1.1.1.1 (Cloudflare) — Быстрая смена DNS, помогает обойти простые блокировки.
  • Temp Mail — Временная почта для регистраций «на один раз».
  • Browserleaks — Проверь свой браузер на утечки (IP, WebRTC, Canvas).

🌐 Защищенные Браузеры#

  • Mullvad Browser — Разработан совместно с Tor Project, но для обычной сети. Лучший против фингерпринтинга.
  • LibreWolf — Форк Firefox с вырезанной телеметрией и встроенной защитой.
  • Brave — Хороший баланс скорости и блокировки рекламы «из коробки».

🧩 OSINT-Расширения (Must Have)#

  • uBlock Origin — Не просто блокировщик рекламы, а мощный фильтр контента.
  • Privacy Badger — Автоматически блокирует невидимые трекеры.
  • Mitaka — Позволяет искать по IP, доменам и хешам в один клик прямо из контекстного меню.
  • SquareX — Одноразовые браузерные вкладки (Sandbox) для открытия подозрительных ссылок.
