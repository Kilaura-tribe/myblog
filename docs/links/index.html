OSINT Ресурсы

🔍 Общий поиск и Никнеймы#

  • Sherlock — Поиск по 400+ сайтам.
  • Maigret — Мощный инструмент для сбора досье по нику.
  • EPIEOS — Разведка по адресу почты.

🇺🇦 Украина (Реестры и Базы)#

  • YouControl — Проверка компаний и ФОП.
  • Opendatabot — Мониторинг официальных данных.
  • Судова влада — Поиск по судебным делам.

📱 Telegram & Соцсети#

  • TGStat — Аналитика каналов и чатов.
  • Telemetr — Детальный анализ Telegram-трафика.
  • Search4Faces — Поиск лиц в ВК и ОК.

📍 Карты и Транспорт#

  • Flightradar24 — Отслеживание самолетов в реальном времени.
  • MarineTraffic — Мониторинг морских судов.
  • SunCalc — Определение времени по теням (для геолокации).

🛡 Инструменты проверки#

  • VirusTotal — Анализ подозрительных файлов и ссылок.
  • URLScan.io — Безопасный просмотр содержимого сайтов.

Warning: При работе с реестрами всегда используй защищенное соединение.

📷 Фото и Видео OSINT#

  • PimEyes — Самый мощный поиск лиц в интернете.
  • Yandex Images — Лучший алгоритм поиска копий фото.
  • Jeffrey’s Image Metadata Viewer — Глубокий анализ EXIF-данных (метаданных) снимка.
