### 🔍 Никнеймы и Почта
* [Sherlock](https://github.com/sherlock-project/sherlock) — Поиск по 400+ сайтам.
* [Maigret](https://github.com/sovereign/maigret) — Глубокий анализ по нику.
* [EPIEOS](https://epieos.com/) — Разведка по Email.

### 🇺🇦 Украина (Реестры)
* [YouControl](https://youcontrol.com.ua/) — Проверка ФОП/Компаний.
* [Opendatabot](https://opendatabot.ua/) — Мониторинг данных.
* [Судова влада](https://court.gov.ua/fms/show_all) — Судебные дела.

### 💰 Криптовалюты (Blockchain OSINT)
* [Blockchain.com](https://www.blockchain.com/explorer) — Эксплорер BTC/ETH.
* [WalletExplorer](https://www.walletexplorer.com/) — Поиск связей между кошельками.
* [Chainabuse](https://www.chainabuse.com/) — База данных мошеннических адресов.

### 🛡 Утечки и Даркнет
* [Have I Been Pwned](https://haveibeenpwned.com/) — Проверка почты на утечки.
* [Snusbase](https://snusbase.com/) — (Premium) Поиск по слитым базам.
* [Onion Engine](https://onionengine.com/) — Поисковик по .onion сайтам.

### 📍 Геолокация
* [SunCalc](https://www.suncalc.org/) — Определение времени по теням.
* [PeakVisor](https://peakvisor.com/) — Идентификация гор по фото.

> **Note:** Всегда используй разные браузерные профили при работе с этими ссылками.

### 🛡 Гигиена и Анонимность
* [Mullvad VPN](https://mullvad.net/) — Самый анонимный (не требует даже почту при регистрации).
* [Proton VPN](https://protonvpn.com/) — Надежный сервис из Швейцарии, есть хорошая бесплатная версия.
* [1.1.1.1 (Cloudflare)](https://1.1.1.1/) — Быстрая смена DNS, помогает обойти простые блокировки.
* [Temp Mail](https://temp-mail.org/) — Временная почта для регистраций «на один раз».
* [Browserleaks](https://browserleaks.com/) — Проверь свой браузер на утечки (IP, WebRTC, Canvas).

### 🌐 Защищенные Браузеры
* [Mullvad Browser](https://mullvad.net/en/browser) — Разработан совместно с Tor Project, но для обычной сети. Лучший против фингерпринтинга.
* [LibreWolf](https://librewolf.net/) — Форк Firefox с вырезанной телеметрией и встроенной защитой.
* [Brave](https://brave.com/) — Хороший баланс скорости и блокировки рекламы «из коробки».

### 🧩 OSINT-Расширения (Must Have)
* [uBlock Origin](https://ublockorigin.com/) — Не просто блокировщик рекламы, а мощный фильтр контента.
* [Privacy Badger](https://privacybadger.org/) — Автоматически блокирует невидимые трекеры.
* [Mitaka](https://github.com/fast-track/mitaka) — Позволяет искать по IP, доменам и хешам в один клик прямо из контекстного меню.
* [SquareX](https://sqrx.com/) — Одноразовые браузерные вкладки (Sandbox) для открытия подозрительных ссылок.
